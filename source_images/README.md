# source_images/

Drop chassis photographs here (`.jpg/.jpeg/.png/.heic/...`). They are **not**
committed — the `.gitignore` keeps the user's own photos and any copyrighted
scans local. The pipeline discovers everything in this directory.

For best provenance results, include close-ups of: the **tube chart**, each
**transformer's part-number / date-code stamp**, the **pot date codes**, and
macro shots of the **eyelet board** sections.
