# sample_schematics/

Reference schematics / layout documents (PDF or image) for the circuits you want
to match against. Contents are **not** committed (copyright + binary bloat); the
distilled, version-controlled ground truth lives in `data/references/*.yaml`,
hand-transcribed from these documents.

First specimen used: `Fender_bassman_6g6b.pdf` (factory layout) and
`Fender_bassman_6g6c.pdf` (factory schematic).
